# Dashboard de gerenciamento predial inteligente

## Para executar

Instale dependências

    
```bash
npm install
```

Crie um arquivo `.env` na raíz do projeto e configure as variáveis de ambiente

```
ENDPOINT=
```

Inicie o servidor

        
```bash
npm run dev
```

Agora você pode visitar https://localhost:3000 no seu navegador.
